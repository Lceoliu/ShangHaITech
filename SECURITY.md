# 安全政策

请不要在公开 issue 中披露敏感漏洞。

如果问题涉及以下内容，请私下联系维护者：

- 账号接管
- 凭据泄露
- 生产 API 行为
- 私有后端 endpoint
- 内部额度、资源调度或风控逻辑
- 用户数据泄露
- 绕过校园系统限制

## 请勿提交

- `.env` 文件
- API key
- JWT secret
- Cloudflare 凭据
- 数据库快照
- 包含用户数据的日志
- 会话文件
- 私有 API 文档
- 生产部署配置

## 支持范围

公开仓库支持社区可见代码和文档。私有生产仓库可能包含不会同步公开的安全修复、生产配置和敏感实现。
