# ShangHaITech: All In One App

ShangHaITech 是一个面向上海科技大学校园生活的 All-in-One App。它尝试把课程、DDL、邮件、公告、Blackboard、AI 工具和常用校园服务整合到一个更轻、更快、更适合手机端的入口里，让学生少切系统、少翻邮件、少错过关键事项。

English version: [README.en.md](README.en.md)

## 为什么值得关注

大学里的信息经常分散在多个系统：课程平台、选课信息、邮件、群通知、DDL、文件下载、评教入口和各种临时公告。ShangHaITech 的目标是把这些碎片化入口整理成一个可信、可扩展、以学生体验为中心的校园助手。

这个项目关注三件事：

- 更快找到信息：把课程、DDL、邮件和常用功能放到统一入口，减少重复登录和来回跳转。
- 更少遗漏事项：用红点、提醒、待处理列表和公告机制，把重要变化推到用户眼前。
- 更适合移动端：针对手机屏幕重新组织交互，而不是简单把网页塞进 App。

## App 内容

已规划和持续完善的能力包括：

- 首页看板：集中展示今日课程、计划日程、公告、消息提醒和关键入口。
- 课程助手：查看学期课程、课程详情、考试安排、课程评价和相关信息。
- DDL 中心：聚合课程作业、Gradescope、Blackboard 等来源的截止日期。
- 智能邮件：同步校园邮箱，提取摘要，按相关性和重要性整理待处理事项。
- Blackboard 工具：面向课程内容、通知、文件和任务的移动端整理入口。
- 智能小 Q / GenAI：提供校园场景下的问答、总结和辅助处理能力。
- 文件中转：降低手机和电脑之间传文件的摩擦。
- 一键评教：简化重复性表单和流程。
- 消息中心：接收系统消息、公告和后续推送提醒。
- 账号与设置：统一管理账号绑定、邮箱绑定、隐私协议和主题偏好。

## QA

**Q: ShangHaITech 是什么？**
A: 一个面向上海科技大学学生的校园生活 App，目标是整合课程、DDL、邮件、公告、AI 工具和常用校园服务。

**Q: 现在能直接安装吗？**
A: 安装包会通过 GitHub Releases 发布。不同版本的可用功能请以 release notes 为准。

**Q: 为什么需要这个 App？**
A: 因为校园信息分散在多个系统里，手机端体验也不统一。ShangHaITech 希望把高频信息和待办事项整理到一个更顺手的入口。

**Q: 会支持更新提醒吗？**
A: 会。后续 APK 版本会通过 GitHub Releases 分发，App 内会根据版本策略提示可选更新或必要更新。

**Q: 可以提功能建议吗？**
A: 可以。建议在 issue 中说明用户场景、期望行为、必要数据、失败状态和隐私影响。

**Q: 项目适合哪些贡献？**
A: 移动端 UI、深色模式、可访问性、文案、本地化、轻量工具、扩展 manifest、测试和文档都适合参与。
